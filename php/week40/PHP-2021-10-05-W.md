Met een `for` loop kun je een bepaald stuk code meerdere keren achter elkaar laten uitvoeren, zo vaak als je wilt. Je maakt een for-loop op de volgende manier:

```php
for ($i = 0; $i < 100; $i++) {
	// Code hier
}
```
